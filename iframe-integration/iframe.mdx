# Iframe Introduction

You can embed Hindsight directly inside your app or dashboard using iframe mode.

## Example

\`\`\`html

\<iframe

  src="https://hindsight.vip/iframe?sim=example"

  width="100%"

  height="700"

  frameborder="0"

  allowfullscreen

\>\</iframe\>
