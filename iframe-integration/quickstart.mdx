# Iframe Quickstart

Get started with iframe integration quickly.

## Steps

- Create a simulation
- Go to Embed section
- Copy the iframe snippet
- Add it to your website

## Example

```html
<iframe src="https://hindsight.vip/eth/token/0xa0b86991c6218b36c1d19d4a2e9eb0ce3606eb48?embed=true" allow="clipboard-write" referrerpolicy="no-referrer-when-downgrade" title="USDC Token Visualization"></iframe>
```
